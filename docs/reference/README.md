## 如何添加技术参考文档

### 文档分类

技术参考文档分为以下几类：

1. **路由与架构**: 系统架构、路由设计
2. **开发工具与调试**: 调试指南、工具使用
3. **技术说明**: 特定技术的说明文档

### 文档模板

```markdown
# 文档标题

## 概述
简要描述文档内容

## 使用场景
何时需要参考本文档

## 详细内容
具体的技术说明、步骤或示例

## 参考资料
相关链接
```

## 维护原则

- 技术参考文档应保持简洁实用
- 避免与设计文档重复
- 定期更新过时内容
- 提供具体的操作示例
